Comentários

os comentários em JavaScript podem ser escritos de duas maneiras

Podemos escever com duas baras "//" para comentar em uma unica linha

Tambem podemos escrever /* */ para comentar em várias linhas

Exemplos de Comentarios em JavaScript

/*
window.alert('Mensagem em Alerta no HTML com JavaScript')
window.confirm('Comando para confirmar algo no HTML com JavaScript')
*/

//window.prompt('Qual é seu nome?')

Notemos que o primeiro comando esta para todas linhas dentro do /* */ e o segundo para somente em uma linha. e assim que funciona os comentarios em JavaScript
