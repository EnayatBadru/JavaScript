Primeiros comandos em JavaScript no curso de Gustavo Guanaraba

window.alert('')

é um comando do JavaScript que serve para alertar uma determinada coisa na pagina HTML

window.confitm('')

é um comando do JavaScript que serve para confirmar uma determinada coisa no HTML usando um comando do JavaScript

window.prompt('')

é um comando do JavaScript que serve para Gerar um espaço, nas aspas simples nos fazemos uma determinada pergunta e, junto com HTML ele vai gerar um campo para escrever a resposta.

Observação

Notemos que os comandos de JavaScript não terminao necessariamente com ponto e vírgu;a ";"

Bom antigamente era obrigatório que os comandos em JavaScript terminasse com ponto e vírgula mas hoje em dia já não é obrigatório, a pessoa pode sim muito bem escrever o ponto e virgula como tambem não as duas formas estão corretas.
